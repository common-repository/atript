Adds the atript script in the head of the website. 

== Description ==

Adding the atript script to your website has never been easier. Simply copy and paste your atript script, that you got from us - and that's it!

For more information visit: [atript](www.atript.org)

== Installation ==

1. Upload `atript_container` directory to the `/wp-content/plugins/` directory or use plugin search - Admin > Plugins > Add new > Search for 'atript_container'.
2. Activate the plugin through the 'Plugins' menu in WordPress 
3. Copy the container-snippet from atript and paste it in the settings page (Admin > Settings > atript Container).
4. Click 'Save' and it's done!

Need help? Contact us if you have any questions regarding [atript](www.atript.org).
